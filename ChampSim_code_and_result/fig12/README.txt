1. This directory has scripts to generate fig12, and the end-to-end bandwidth for the LeakyRand channel, including the CRFill and CRProbe overhead for 1 MB, 2 MB, and 4 MB LLC sizes. The bandwidth results are mentioned in section5.9 in paper.
2. ./script.sh will start the simulations.
3. set ${num_prl} as per your machine. The value for ${num_prl} should be less than the number of logical cores available on your machine.
